This directory should contain all images for the website.


## Image Requirements

- **House Photos**: JPEG format, recommended size 1200x800px
- **Floor Plans**: PNG format with transparent background
- **Logo**: PNG format with transparent background
- **Index Slider**: JPEG format, recommended size 1920x1080px

## Note

The actual image files are not included in the repository. 
You need to add your own images to these folders for the site to display properly.
