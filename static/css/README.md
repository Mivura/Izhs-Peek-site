# Static CSS Directory

This directory should contain all CSS files for the website.

## Required Files

### Created by Refactoring:
- ✅ `house.css` - Modern, optimized styles for house pages (created)

### Need to be Added:
The following CSS files are referenced by templates but not yet in the repository:
- `style.css` - Main site styles
- `styleForZaya.css` - Contact form styles
- `style_house.css` - Legacy house styles (may be replaced by house.css)
- `catalog.css` - Catalog page styles
- `about_style.css` - About page styles

## Note

These CSS files may already exist in your project but are not tracked in git.
You need to add them from your local development environment or create new ones.

The refactoring introduces `house.css` which provides modern, mobile-responsive styles for house pages with optimized sliders.
