# Руководство по редактированию характеристик и комплектаций домов

## Введение

При добавлении нового дома вы можете настраивать различные параметры в двух местах:
1. **`houses_data.json`** - для данных, отображаемых на главной странице и в каталоге
2. **`templates/domN_kark/house_1.html`** - для детальной информации на странице дома

---

## Часть 1: Редактирование houses_data.json

### Что можно редактировать в JSON

Откройте файл `houses_data.json` и найдите или добавьте запись для вашего дома:

```json
{
  "id": 6,
  "name": "Ваше название дома",           // ← Название дома
  "route": "house_kark_6",                // ← Имя функции маршрута в app.py
  "template": "dom6_kark/house_1.html",   // ← Путь к шаблону
  "image_folder": "dom6_kark",            // ← Папка с изображениями
  "main_image": "dom6_kark/1.jpg",        // ← Главное фото для каталога
  "description": "Краткое описание дома", // ← Описание в каталоге
  "area": "250 м²",                       // ← Общая площадь
  "terrace": "70 м²",                     // ← Площадь террасы
  "size": "13×18 м",                      // ← Размеры дома
  "price": "от 15 000 000 ₽",            // ← Базовая цена
  
  "characteristics": {
    "total_area": "250 м²",               // ← Общая площадь (детально)
    "living_area": "200 м²",              // ← Жилая площадь
    "terrace_area": "70 м²",              // ← Площадь террасы
    "floors": "2 этажа",                  // ← Количество этажей
    "bedrooms": "5 спален",               // ← Количество спален
    "bathrooms": "3 санузла"              // ← Количество санузлов
  },
  
  "photos": [                             // ← Список фотографий для слайдера
    "/1.jpg",
    "/2.jpg",
    "/3.jpg",
    "/4.jpg"
  ],
  
  "plans": [                              // ← Планировки этажей
    {
      "image": "dom6_kark/plan.png",
      "title": "1 этаж"
    },
    {
      "image": "dom6_kark/plan2.png",
      "title": "2 этаж"
    }
  ]
}
```

### Параметры в JSON (подробно)

| Параметр | Где отображается | Пример значения |
|----------|------------------|-----------------|
| `name` | Главная страница, Каталог | "Скандинавский дом" |
| `description` | Каталог | "Современный дом с панорамными окнами" |
| `area` | Главная страница (карточка) | "250 м²" |
| `terrace` | Главная страница (карточка) | "70 м²" |
| `size` | Главная страница (карточка) | "13×18 м" |
| `price` | Главная страница (карточка) | "от 15 000 000 ₽" |
| `characteristics.*` | Страница дома (секция характеристик) | См. выше |
| `photos` | Страница дома (слайдер фотографий) | Массив путей к фото |
| `plans` | Страница дома (слайдер планировок) | Массив планов этажей |

### 2.2 Таблица комплектаций

Найдите секцию **"КОМПЛЕКТАЦИИ ДОМА"** (примерно строки 485-605):

```html
<h1 class="complectation-title">КОМПЛЕКТАЦИИ ДОМА</h1>

<table class="complectation-table">
    <thead>
    <tr>
        <th>Характеристики</th>
        <th>Теплый контур</th>      <!-- ← Можно переименовать -->
        <th>Инженерный пакет</th>    <!-- ← Можно переименовать -->
        <th>Чистовая отделка</th>    <!-- ← Можно переименовать -->
    </tr>
    </thead>
    <tbody>
    <!-- Строка комплектации -->
    <tr>
        <td>
            <div class="feature-name">Фундамент</div>        <!-- ← Название -->
            <div class="feature-details">                     <!-- ← Детали -->
                Монолитная железобетонная плита 200мм
            </div>
        </td>
        <td class="checkmark">✓</td>  <!-- ← Включено в пакет 1 -->
        <td class="checkmark">✓</td>  <!-- ← Включено в пакет 2 -->
        <td class="checkmark">✓</td>  <!-- ← Включено в пакет 3 -->
    </tr>
    
    <!-- Добавьте больше строк по аналогии -->
    </tbody>
</table>
```

#### Как редактировать комплектации:

**Изменить название пакета:**
```html
<th>Базовый пакет</th>  <!-- Вместо "Теплый контур" -->
```

**Добавить новую характеристику:**
```html
<tr>
    <td>
        <div class="feature-name">Утепление</div>
        <div class="feature-details">Минеральная вата 200мм</div>
    </td>
    <td class="checkmark">✓</td>
    <td class="checkmark">✓</td>
    <td class="checkmark">✓</td>
</tr>
```

**Убрать галочку (не включено в пакет):**
```html
<td></td>  <!-- Пустая ячейка вместо <td class="checkmark">✓</td> -->
```

**Изменить детали:**
```html
<div class="feature-details">
    Ваш новый текст описания<br>
    Можно добавить несколько строк
</div>
```

## Часть 3: Калькулятор (опционально)

Найдите секцию **"КАЛЬКУЛЯТОР СТРОИТЕЛЬСТВА ДОМА"** (примерно строки 950+):

### 3.1 Базовые комплектации калькулятора

```html
<div class="config-option" data-price="1200000">
    <div class="config-name">Сруб из бруса 150x150мм</div>      <!-- ← Название -->
    <div class="config-description">                            <!-- ← Описание -->
        Базовый комплект: коробка дома, кровля
    </div>
    <div class="config-price">1 200 000 ₽</div>                <!-- ← Цена -->
</div>
```

**Что изменить:**
- `data-price` - цена в рублях (без пробелов)
- `config-name` - название комплектации
- `config-description` - описание
- `config-price` - отображаемая цена (с пробелами и символом ₽)

### 3.2 Дополнительные услуги

```html
<div class="service-item" data-price="132000">
    <input type="checkbox" name="services" 
           value="Свайно-винтовой фундамент" class="service-checkbox" id="service1">
    <div class="service-info">
        <div class="service-title">Свайно-винтовой фундамент 16 свай</div>
        <div class="service-description">• проба грунта</div>
        <div class="service-price">132 000 ₽</div>
    </div>
</div>
```

**Что изменить:**
- `data-price` - цена услуги (для расчета)
- `value` - название для отправки формы
- `id` - уникальный идентификатор (service1, service2 и т.д.)
- `service-title` - название услуги
- `service-description` - описание
- `service-price` - отображаемая цена

## Быстрая шпаргалка

### Добавление нового дома - чек-лист:

1. **Создать папки:**
   - `static/img/dom6_kark/` - для изображений
   - `templates/dom6_kark/` - для шаблона

2. **Добавить изображения:**
   - `1.jpg`, `2.jpg`, `3.jpg` - фотографии
   - `plan.png` - планировка

3. **Создать запись в `houses_data.json`:**
   - ID, название, маршрут
   - Характеристики, фото, планы
   - Цены

4. **Скопировать и отредактировать шаблон:**
   - `cp templates/dom1_kark/house_1.html templates/dom6_kark/house_1.html`
   - Заменить `dom1_kark` на `dom6_kark` во всех путях
   - Изменить характеристики (строки 140-175)
   - Изменить комплектации (строки 485-605)
   - Изменить цены (строки 583-600)

5. **Добавить маршрут в `app.py`:**
```python
@app.route('/house6')
def house_kark_6():
    """Страница дома 6"""
    houses = load_houses_data()
    house = next((h for h in houses if h['id'] == 6), None)
    photos = house['photos'] if house else []
    return render_template("dom6_kark/house_1.html", photos=photos, house=house)
```

6. **Перезапустить приложение:**
```bash
python3 app.py
```

## Важные замечания

⚠️ **При изменении характеристик:**
- Обновляйте данные и в JSON, и в HTML шаблоне
- Сохраняйте согласованность между разными местами

⚠️ **При изменении цен:**
- Базовая цена в JSON отображается в каталоге
- Детальные цены комплектаций - в HTML шаблоне

⚠️ **При работе с калькулятором:**
- Атрибут `data-price` должен содержать число БЕЗ пробелов
- Отображаемая цена может содержать пробелы и символы

⚠️ **После изменений:**
- Всегда перезапускайте Flask приложение
- Проверяйте результат в браузере
